Etiquetas

Html es un editor de texto. Dicho editor de texto trabaja con etiquetas.

Las etiquetas son marcas con las que el procesador de la computadora trabaja.

Todas las etiquetas siempre se escriben < nombredeetiqueta > sin espacio y al finalizar se le agrega la misma etiqueta pero con un cierre (/) < /nombredetiqueta >.

Todas estas etiquetas vienen pre-establecidas con ciertos fomas y formatos (uno puede sacarles todas esas formas y formatos) y cada una esta orientada a trabajar con un tipo de forma y formato especifico.

Hay diferentes categorias de etiquetas. Tenemos las categorìas de etiquetas semanticas y las no semanticas. Otra categoria de etiquetas son de bloque y de Inline o de ambas al mimso tiempo. Y luego tenemos las etiquetas de atributos.

Etiquetas y usos.

    Etiquetas de Bloques

    Las etiquetas de bloque son aquellas que generan una especie de contenedor rectangular especifico y que tiene ciertos atributos determinados

    Dentro de ellas se puede diferenciar las semanticas y las no semanticas.

    Solo se utiliza las no semanticas cuando no hay alguna etiqueta semantica que pueda ser utilizada.

  • <head> se utliza antes que el la etiqueta body. En dicha etiqueta se dan los links necesarios y a programas a los que se les tenga que hacer referencia antes de entrar en el cuerpo del articulo.
  • <body> es el cuerpo del programa donde se pone la información de interes para el que este leyendo la pagina.
  • <header> es el elemento que se define como cabecera de la pagina, en este se pone info relacionada a relaciones externas o propias de la pagina, no necesitan ser parte de lo que el main.
  • <main> es la secion principal del articulo o progama
  • <footer> es la seccion que se encuentra en el pie de pagina del documento.
  • <aside> es una seccion del articulo secundaria.
  • <section> es la etiqueta que te permite dividir en secciones el articulo de tu pagina para poder con cada una de ellas de forma independiente al resto.
  • <style> es la etiqueta de estilo que se utiliza para dar un estilo de texto a un bloque.
  • <article> ver significado.
  • <time> ver el significado.
  • <a> ancla un objeto en la ventana del programa.
  • <link> es una etiqueta que se utiliza para CSS. Dicha etiqueta da referencia a un lugar por fuera del archivo de origen.
  • <img> es una etiqueta que hace referencia a una imagen.
  • <p> se utiliza para generar un parrafo.
  • <ul> se utiliza para genera una lista sin orden.
  • <ol> se utiliza para realizar una lista ordenada.
  • Ejemplo de lista ordenada:
    1. Item.
    2. Item.
  • <form> se utiliza para crear un formulario interactivo.
  • <time> no se de que se trata.
  • <nav> es para dar urls.
  • <div> esta es la unica etiqueta no semantica, ella no tiene ninguna forma ni formato especifico. Se comporta igual que el resto como de bloque.

    Etiquetas de Texto

    Las etiquetas de texto trabajan sobre el texto directamente.

  • <del> se utiliza para tachar el texto escrito. Ej.:Tachado.
  • <u> se utiliza para subrayar la linea. Ej.: Subrayado
  • <strong> es una etiqueta que se utiliza para remarcar una palabra o frase. Ej:Remarcada de frase
  • <em> es una etiqueta que se utiliza para enfatizar un texto. Ej:Texto enfatizado
  • <mark> es una etiqueta para remarcar texto. Ej:Texto remcarcado
  • <cite> es una etiqueta para dar formato de cita. Ej:Anonimo
  • <abbr> es una etiqueta para abreviatura de palabras y permite leer al pararse sobre el texto la palabra o frase completa. Ej: TBBG
  • <span> es la etiqueta de texto sin formato, no semantica. Con ella se puede dar el estilo que se quiera al texto. Ej: todos las etiquetas de texto anterior. (ojo algunos formatos se pisan)
  • A los colores al ser ingresados con style="color:#......", el valor dado es hexadecimal y dicho dato es el valor del mismo, se puede utilizar el metodo RGB y el RGBA

Referencias:

Hay diferentes tipos de referencias

  • Referencias Internas: Aquellas que recorren el mimso sitio o espacio donde te encotras (querer moverse de una punta a la otra de la misma pagina y/o a un sitio Indexado por la misma pagina).
  • Referencias Externas: Aquellas que te llevan a diferentes enlaces fuera de dicho sitio tanto de la internet como otros archivos de la compu (te llevan del sitio web que te encotras a otro distinto).

Atributos

Los atributos son las cosas que se les puede dar a las diferentes etiquetas o elementos.

  • <elemento href=""> es un atributo de referencia a la posicion especifica donde se encuentra el enlace al que se le dio. Puede ser tanto Interna al mismo programa como externa.
  • <elemento rel=""> indica cual es la relación entre el elemento y el atributo.
  • <elemento class=""> le da un nombre de clase al elemento para luego poder ser llamado por esa clase.
  • <elemento id=""> le da un nombre de identificacion para luego poder ser llamado por ese nombre.
  • <elemento style=""> este atributo (que se puede usar como etiqueta) es el que permite modificar la forma en que se ve el texto. desde el html. Dicho sea de paso los estilos ya no se trabajan más dentro del html sino que se trabajan en una hoja aparte de estilos css.

Los atributos CSS

Los atributos en Css generan lo mismo que si uno los pusiera en el html. Esta separacion se realiza para que quede el estilo de lo que uno ve en una hoja separada a la información de lo que uno esta viendo.

De este momento en adelante se va a realizar de la siguiente manera el llamado a el elemento o etiqueta a dar estilo. Selector este llama a nombre de bloque o se elementos. .Selector este llama a aquellos elementos que esten dentro de dicha clase. #Selector y este llama a aquellos que fueron identificados con dicho nombre.

Importante para poder llamar a un elemento que esta dentro de un bloque se realiza de la siguiente manera: Selector Selector {aqui dentro van las propiedades del elemento}

  • width: atributo que permite darle un ancho determinado al bloque.
  • alt: permite darle una definicion al elemento cuando se pasa por encima del mismo.
  • height: permite darle un altura determinada al bloque.
  • type: permite ponerle un tipo al elemento. Este atributo es precizado en los formularios.
  • border: permite generar un borde a la estructura del elemento.
  • float: permite poner el elemento en el vertice z por delante de otros elementos generando. Puede ser para la izquierda como para la derecha y para desactivar esta flotación en el siguiente elemento u bolque se debe de poner clear y dicho lado en que se lo puso
  • overflow: se puso para que cuando un bloque no tiene en si mismo nada mas que otro, el bloque ocupe el espacio de todo el bloque subsiguiente.
  • z-index: es un atributo que permite poner en un determinado punto de un indice z (para aquellos bloques flotantes pone uno por encima del otro).
  • margin es un atributo que le da un margen al elemento, para poder despegarlo de forma tal que no este pegado al siguiente elmento del mismo bloque. Se le puede poner margen a cualquiera de los cuatro lados del bloque.
